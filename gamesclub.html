Games Club

Current Game

The current game is OpenTTD. It is an old school "business simulation game" but I'd describe it more as a transport logistics game. It's a free open-source continuation of the original Transport Tycoon Deluxe that adds many modern improvements as well as a robust mod management system.

The Game's Club is invite-only. If you're not already there, ask someone who is.

How to Play

OpenTTD is free and available for pretty much any platform you'd play games on. It should run on nearly anything. You can find it on the official website, Steam, GOG, and probably a bunch of other places. We're playing on the newest version, which is 13.0-beta1. If you got the game from Steam, switch to the "Testing" beta. As far as the mods used on the server, there's nothing you need to do! The game will automatically load mods and apply the settings the server asks for.

Mods/Tweaks on the Server

Since I know many players are newer, we will be playing a vanilla+ game. This means that the above tutorials should be mostly correct. The important settings that we are tweaking, and their effects, are as follows:

  • Cargodist - This is probably the most impactful one, though I think the change is fairly intuitive. Basically, mail and passengers will have specific destinations. In the base game, they can be sent anywhere. This just encourages more realistic passenger transport. For more details look here. I have both set to "symmetric".
  • When paused allow all actions - so you can build even if the game is paused.

Additionally we'll be using the following mods:

  • OpenGFX+ mods - These are basically just extensions of the base game, adding some more small features and world generation options.
  • eGRVTS 2.1, av8, and Squid ate FISH - These all add more vehicles to the game.
  • Midwestern City Names - to make it feel more like home
